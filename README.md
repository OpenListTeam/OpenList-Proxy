# OpenList-Proxy

Use another machine to proxy OpenList's traffic.

- [x] Cloudflare Workers
- [x] Golang

## Usage

```shell
Usage of OpenList-Proxy:
  -address string
        openlist address
  -cert string
        cert file (default "server.crt")
  -help
        show help
  -https
        use https protocol.
  -key string
        key file (default "server.key")
  -port int
        the proxy port. (default 5243)
  -token string
        openlist token
  -version
        show version and exit
```
